# Roblox Site

Це простий веб-сайт про гру Roblox, створений з використанням Flask, HTML та CSS.

## Структура проекту

- `app.py`: Головний файл Flask додатку
- `templates/`: Шаблони HTML
  - `base.html`: Базовий шаблон
  - `index.html`: Головна сторінка
  - `about.html`: Сторінка "Про нас"
  - `contacts.html`: Сторінка контактів
- `static/css/style.css`: Стилі CSS

## Як запустити

1. Переконайтеся, що у вас встановлений Python та Flask.
2. Запустіть додаток командою: `python app.py`
3. Відкрийте браузер і перейдіть на `http://127.0.0.1:5000`

## Сторінки

- **Головна**: Опис гри Roblox та її переваги
- **Про нас**: Інформація про сайт та команду
- **Контакти**: Форма для зв'язку

Дизайн сучасний та мінімалістичний, підходить для вивчення Python та веб-розробки.
